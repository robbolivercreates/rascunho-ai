# Rascunho AI

Canvas inteligente com IA — powered by Gemini 3.1 Flash-Lite.

## Features
- 🎠 Gerador de carrosséis no canvas (Dicas, Mind Map, Passos, Comparação)
- 🐦 Thread X/Twitter style
- 🤖 AI Backend com Gemini (diagramas, SVG, carrosséis)
- 🎨 Paletas dark de alto engajamento

## Setup
```bash
cd ai-backend
GEMINI_API_KEY=your_key node server.js
```

## Stack
- Frontend: Excalidraw (fork customizado)
- Backend: Node.js + Express + Gemini API
- Renderer: Excalidraw native elements
